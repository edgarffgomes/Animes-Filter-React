# Anime Filter React
> Status: Completo ✅
> 
### Uma Single Page Aplication que lista animes a partir de uma busca feita pelo usuário.

## Funcionalidades implementadas:
+ Consulta à API
+ Retorno de buscas à API
+ Listagem de objetos
+ Otimização de tempo de resposta das pesquisas
